A demonstration of how to use [GTK+](http://www.gtk.org/) with [Go](http://golang.org/)

## Install

    go get github.com/pebbe/go-gtk-demo


This uses GTK+ 2

If you want to use GTK+ 3, change the fourth line in go-gtk-demo.go
